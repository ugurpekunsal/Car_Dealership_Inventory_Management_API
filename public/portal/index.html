Car Dealership API Documentation

Authentication

All API calls require an API key in the header:

x-api-key: your_api_key_here

Endpoints

GET /car-dealership-api/cars

Get all cars

Example Response:

[
    {
        "id": "CAR_123",
        "make": "Toyota",
        "model": "Camry",
        "year": 2022,
        "price": 25000,
        "status": "available"
    }
]
GET /car-dealership-api/cars/{id}

Get car by ID

POST /car-dealership-api/cars

Create a new car

Request Body:

{
    "make": "Toyota",
    "model": "Camry",
    "year": 2022,
    "price": 25000
}
PUT /car-dealership-api/cars/{id}

Update a car

PATCH /car-dealership-api/cars/{id}

Partially update a car

DELETE /car-dealership-api/cars/{id}

Delete a car

Code Examples

JavaScript

// Get all cars
const response = await fetch('/car-dealership-api/cars', {
    headers: {
        'x-api-key': 'your_api_key_here'
    }
});
const cars = await response.json();

Python

import requests

headers = {
    'x-api-key': 'your_api_key_here'
}
response = requests.get('http://your-api-url/car-dealership-api/cars', headers=headers)
cars = response.json()
